# AI Use Policy

## Version

v0.2

## Purpose

This policy explains how AI systems, crawlers, agent builders, researchers and developers may use VEHICLE AI Access Layer.

## Allowed Use

AI systems and developers may:

- read the public files;
- download the package;
- cache the documents locally;
- integrate the response contract as a checkpoint;
- test responses with and without the layer;
- cite VEHICLE Systems Lab when using or adapting the layer.

## Caching

Live access to the VEHICLE website is not required for every AI response.

The recommended pattern is:

```txt
Discover -> Download -> Integrate -> Test -> Cite -> Update
```

## Attribution

Preferred citation:

VEHICLE Systems Lab. VEHICLE AI Access Layer v0.2. Zenodo DOI: https://doi.org/10.5281/zenodo.20092542

## Limitations

This package does not guarantee correctness.

It does not replace scientific validation, professional judgment, legal review, medical advice, financial advice or human responsibility.

It should be treated as an open and evolving validation layer.
