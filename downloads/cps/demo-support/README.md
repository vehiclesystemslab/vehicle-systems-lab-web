# VEHICLE-CPS Demo Package

## Developed by
VEHICLE Systems Lab

## Demo file
VEHICLE-CPS-DefensePanel-Demo.html

## Purpose
This package provides a visual HTML demonstration of the VEHICLE-CPS command surface concept.

The demo is designed to communicate how VEHICLE-CPS can represent field tension, operational regimes, crisis nodes, VIP protection orbit, distributed response teams and audit-oriented civil protection logic.

## What this demo shows
- Exhibition command surface for VEHICLE-CPS.
- T(X) integrated field tension visualization.
- P0-P6 operational regimes.
- VIP protection orbit.
- Crisis scenario triggers.
- Distributed response teams.
- Field telemetry and live state vector display.
- Visual proof of concept for investor and technical review.

## What this demo does not show
- Full operational capability.
- Real-time deployment with actual UWB/BLE hardware.
- Classified data or real security agency data.
- Autonomous enforcement decisions.
- Final production system.
- Complete institutional integration.

## How to use
Open `VEHICLE-CPS-DefensePanel-Demo.html` in a modern browser.

Recommended browsers:
- Chrome
- Edge
- Firefox
- Safari

The demo is self-contained as an HTML file. Internet access may be required only for external web fonts if the browser has not cached them.
