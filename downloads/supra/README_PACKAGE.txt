# VEHICLE-SUPRA Investor Package v1

Developed by VEHICLE Systems Lab.

This package contains public investor and technical materials for VEHICLE-SUPRA, an experimental architecture for autonomous systems, lunar-oriented models, extreme environments and advanced operational interfaces.

Recommended public order:
1. VEHICLE-SUPRA-Funding-Brief.pdf
2. VEHICLE-SUPRA-Technical-Brief.pdf
3. VEHICLE-SUPRA-Demo-Package.zip
4. VEHICLE-SUPRA-Operational-Costs.pdf
5. vehicle-supra-ai-reference.txt

The demo is a premium HTML lunar command surface and should be presented as a research demonstration, not as a deployed operational system.
