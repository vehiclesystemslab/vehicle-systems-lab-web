# VEHICLE-ODI Demo Package

Developed by VEHICLE Systems Lab.

## Purpose

This package provides a visual HTML demonstration of the VEHICLE-ODI orbital debris intelligence architecture.

## What This Demo Shows

- Orbital debris decision surface
- Orbital Tension Index (OTI)
- T_ext relational tension
- T_int mass/energy tension
- Orbital bands
- Borda Milan A1-A6 taxonomy
- Extraction priority queue
- Crisis scenarios including fragmentation, conjunction, Kessler cascade and A6 debris cloud

## What This Demo Does Not Show

- Full operational space situational awareness capability
- Real-time satellite tracking
- Classified or live orbital data
- Autonomous debris removal authorization
- Final production system

## How to Use

Open `VEHICLE-ODI-Orbital-Debris-Decision-Surface-Demo.html` in a modern browser.
